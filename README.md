# tweeter_api

## como rodar
Na raiz do projeto executar

- Criar um ambiente virtual
 `python3 -m venv venv` 

- Executar ambiente virtual
`source ./venv/bin/activate`

- Baixar pacotes necesarios 
`pip install -r requirements.txt`
